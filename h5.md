## a) Fuzzzz. Ratkaise dirfuz-1 artikkelista Karvinen 2023: Find Hidden Web Directories - Fuzz URLs with ffuf.
latasin ensin dirfuz-1 komennolla

```
$ wget https://terokarvinen.com/2023/fuzz-urls-find-hidden-directories/dirfuzt-1
```
latasin myös ffuf:in komenolla

```
$ sudo apt-get update
$ sudo apt-get install ffuf
```
Latasin myös sana listan yleisillä reiteillä komennolla

```
$ wget https://raw.githubusercontent.com/danielmiessler/SecLists/master/Discovery/Web-Content/common.txt
```

Annoin ensin oikeudet dirfuzt-1 että se voisi toimia komenolla

```
$ chmod u+x dirfuzt-1
```
seuravaksi ajoin seuraavan komennon ffuf:illa
````
$ ffuf -w ~/common.txt -u http://127.0.0.2:8000/FUZZ -ac
````
tällä sain tuloksen
