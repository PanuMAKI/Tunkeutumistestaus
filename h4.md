# H4 kotitehtävä

## a)
Asennetu OWASP ZAP kali linuxin komenolla "sudo apt-get install zaproxy"
tämän jälkeen käynnistetty ohjelma ja ladattu sertifikaatti joka laitettu firefox selaimeen

Fire foxissa laitettu asetuksista "Manual proxy configuration" asetus ja laitettu http proxyyn ip 127.0.0.1 ja portti 8080.

tämän jälkeen ZAP proxyyn alkoi tulla get komentoja kun selainta käytti.
